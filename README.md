# Restaurant Map Frontend
VusJS + Bootstrap 4 + Google MAP API
### Usage
```
Project setup
- git clone https://github.com/rawinby/restaurant-map-frontend.git
- yarn install
Compiles and hot-reloads for development
- yarn run serve
Compiles and minifies for production
- yarn run build
```
